# Hjälp och guider

Notarkiv är ett internt katalogsystem för vårt notförråd. Här samlar vi
guider och förklaringar uppdelade per roll - välj den som matchar din
behörighet eller läs vilken som helst för att förstå systemet bättre.

## Vad är notarkiv?

- **Katalog över alla noter** i vårt förråd, med titlar, kompositörer,
  arrangörer, besättning och liknande metadata
- **Var ligger noten?** Lagringsplatser (rum, system) och enheter
  (hyllor, pärmar, mappar) som du själv kan söka och bläddra i
- **Utlåning** med kundvagn för att lista alla noter du vill låna inför
  ett tillfälle, sedan checkout som registrerar utlånet och hjälper dig
  plocka
- **Inventering** för att verifiera att lagerlistan stämmer med verkligheten
- **Skanning** av nya noter med automatisk metadataextraktion via OCR

## Hur kommer jag in?

Du loggar in via `/login` med ditt användarnamn och lösenord du fått av
admin. Första gången tvingas du byta lösenord. När du är inloggad ser du
funktioner som matchar din roll.

## Roller

| Roll | Vad det betyder |
|------|-----------------|
| **reader** | Musiker och körledare - kan söka och låna, men inte ändra metadata |
| **editor** | Betrodd personal - kan skanna nya noter, redigera metadata, hantera taggar |
| **admin** | Bibliotekarier - allt + användarhantering, inställningar, lagringsplatser |
